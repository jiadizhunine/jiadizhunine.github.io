# jiadizhunine.github.io

[@jiadizhunine](https://github.com/jiadizhunine) 的 GitHub Pages 个人主页。

网站基于暖色、极简的
[Luka Homepage Template](https://github.com/wzsyyh/luka-homepage-template)
定制，使用纯静态 HTML、CSS 和 JavaScript，无构建步骤，也不包含访客追踪组件。

## 本地预览

```bash
python3 -m http.server 4173
```

随后访问 <http://127.0.0.1:4173/>。
