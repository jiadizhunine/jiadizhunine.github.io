# jiadizhunine.github.io

Personal GitHub Pages site for [@jiadizhunine](https://github.com/jiadizhunine).

The site is a privacy-conscious adaptation of the warm, minimal
[Luka Homepage Template](https://github.com/wzsyyh/luka-homepage-template).
It uses static HTML, CSS, and JavaScript with no build step and no visitor-tracking widget.

## Local preview

```bash
python3 -m http.server 4173
```

Then open <http://127.0.0.1:4173/>.
